## 개요

[[코드 포인트]]는 문자 체계 안에서 특정 위치를 가리키는 숫자 값이다. Unicode에서는 보통 `U+0041`처럼 `U+`와 16진수로 표기하며, `U+0041`은 라틴 대문자 `A`에 대응한다.

## 문자와의 차이

코드 포인트는 "문자 하나"와 항상 같지 않다. Unicode에서 encoded character는 추상 문자와 코드 포인트의 대응이지만, 어떤 추상 문자는 여러 코드 포인트 시퀀스로 표현될 수 있고, 어떤 사용자 지각 문자는 여러 코드 포인트가 합쳐져 만들어질 수 있다.

[[Unicode scalar value]]는 모든 코드 포인트 중 surrogate code point를 제외한 값이다. UTF-8과 UTF-16 같은 Unicode encoding form은 scalar value를 [[코드 유닛]] 시퀀스로 대응시킨다. 따라서 코드 포인트와 scalar value는 동의어가 아니며, surrogate 값 자체는 독립적인 문자 값으로 인코딩되어서는 안 된다.

## 실무적 의미

문자열 처리를 할 때 코드 포인트 수, 코드 유닛 수, 바이트 수, [[그래핌 클러스터]] 수를 구분해야 한다. 예를 들어 한글, 결합 부호, 이모지 시퀀스는 화면에서 한 글자처럼 보이더라도 내부 표현 단위가 여러 개일 수 있다.
