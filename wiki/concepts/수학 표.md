## 개요

[[수학 표]](Mathematical tables)는 반복해서 필요한 함수나 계산의 결과를 미리 구해 입력값 또는 조건과 대응하도록 규칙적인 순서나 행·열 구조로 정리한 참조 도구다. 사용자는 계산 전체를 매번 다시 수행하는 대신 필요한 값을 찾아 쓰고, 정확히 일치하는 값이 없으면 가까운 항목 사이를 보간(interpolation)할 수 있다.

여기서 수학 표는 일반적인 데이터 표나 행렬을 모두 가리키지 않는다. 또한 계산값을 순서대로 나열한 목록도 수학 표가 될 수 있으므로, 계산표라는 개념과 특정한 표 형식(tabular formatting)을 동일시해서는 안 된다. 핵심은 계산 결과를 저장해 다른 계산이나 실무에서 재사용한다는 데 있다. 전자 계산기 이전에는 표 한 권이 한 명 또는 여러 계산자가 수행한 계산 노동을 여러 사용자에게 전달하는 매체였으며, 계산 능력을 개인의 암산이나 필산에서 공유 가능한 자원으로 확장했다.

## 가장 이른 현존 수학 표

현재 알려진 사례 가운데 연대를 확실하게 정할 수 있는 가장 이른 현존 수학 표는 고대 수메르 도시 슈루팍(Šuruppag)에서 나온 점토판이다. 이 유물은 박물관 번호 `VAT 12593`, 발굴·문헌 식별자 `SF 082`, CDLI 번호 `P010678`로 확인할 수 있으며, 초기 왕조 IIIa기인 기원전 약 2600–2500년으로 편년된다. 같은 길이값 두 개와 그 곱으로 얻은 정사각형의 넓이를 대응시키는 계측 표(metrological table)로 해석되며, 계산 결과를 조회 가능한 순서로 저장하는 특징을 보여준다.

> [!NOTE] “최초”의 범위
> 이 점토판을 인류가 실제로 처음 만든 수학 표라고 단정할 수는 없다. 발견되지 않았거나 보존되지 않은 더 이른 자료가 있을 수 있으므로, 이 문서는 “현재 알려진 것 가운데 연대를 확실하게 정할 수 있는 가장 이른 현존 사례”로 표현한다.

이 사례는 수학 표가 근대 과학의 발명품이 아님을 보여준다. 다만 출토 맥락의 일부가 소실되어 실제 사용 목적을 확정하기 어렵고, 표의 구조만으로 토지 측량에 직접 사용됐다고 단정해서도 안 된다. 수메르의 면적표에서 근대 유럽의 로그표까지 하나의 직접 계보가 이어졌다고 해석할 수도 없다. 서로 다른 사회는 계측, 회계, 천문 관측과 교육 같은 필요에 맞춰 서로 다른 표를 만들었다.

## 수학 표의 확산

수학 표의 역사에는 하나의 직선적 발전 단계보다, 반복 계산을 저장하고 공유하려는 여러 전통이 겹쳐 있다.

| 시대와 맥락 | 대표적인 표 | 기능 |
|---|---|---|
| 고대 메소포타미아 | 면적표, 도량형표, 곱셈·역수 계열 표 | 토지와 물자의 수량화, 서기관 교육과 계산 보조 |
| 고대·중세의 천문학 | 천문표, 삼각법 관련 표 | 천체 위치와 시각 계산, 달력과 관측 보조 |
| 근대 초기 과학 | 로그표, 삼각함수표 | 긴 곱셈·나눗셈과 각도 계산의 부담 감소 |
| 18–19세기 국가·산업 실무 | 항해표, 천문표, 생명표 | 항해, 측량, 보험과 행정에 필요한 반복 계산의 표준화 |
| 20세기 응용과학 | 통계표, 특수함수표 | 과학·공학 계산에서 함수값과 확률값 제공 |

이 구분은 각 시대의 모든 표를 포괄하지 않는다. 중요한 공통점은 계산 비용을 한 번 집중해서 지불한 뒤, 정리된 결과를 여러 사람이 반복해서 사용한다는 점이다.

## 범위별로 널리 쓰인 수학 표

이 문서에서 확인한 자료만으로 모든 시대와 지역을 합친 단일 배포 순위를 확정하기는 어렵다. “가장 흔함”은 어느 시대와 지역에서 누구의 어떤 계산을 기준으로 삼는지에 따라 달라진다.

| 비교 범위 | 대표적인 답 | 이유 |
|---|---|---|
| 고대 메소포타미아의 서기 교육 | 곱셈표·역수 계열 표 | 반복해서 쓰는 수 관계를 학습하고 계산에 재사용하게 했다. |
| 컴퓨터 이전의 과학·공학 전문 계산 | 상용로그표와 삼각함수표 | 곱셈·나눗셈과 각도 계산을 조회와 더 단순한 연산으로 바꿨다. |
| 천문학과 항해 | 천문표와 항해표 | 시각, 천체 위치와 항로 계산에 필요한 값을 제공한다. |
| 20세기 응용수학 | 특수함수표와 통계표 | 손으로 직접 계산하기 어려운 함수값과 확률값을 제공한다. |

확인된 자료를 바탕으로 하면 곱셈표·역수 계열 표는 고대 메소포타미아의 서기 교육에서 반복 학습된 사례이고, 상용로그표와 삼각함수표는 컴퓨터 이전 전문 계산에서 여러 분야가 의존한 사례다. 이를 전 시대·전 지역의 절대적인 1위와 2위로 읽어서는 안 된다. [[찰스 배비지]]가 활동하던 19세기 영국의 문제를 이해할 때는 특정 표 하나보다 천문학·항해·공학에서 요구한 고정밀 표의 계산과 인쇄가 핵심이다.

## 표를 이용한 계산

수학 표의 사용은 단순한 복사에 그치지 않는다. 사용자는 일반적으로 다음 과정을 따른다.

1. 문제에서 필요한 함수와 입력값을 정한다.
2. 표에서 입력값과 일치하거나 가까운 행과 열을 찾는다.
3. 필요한 경우 자릿값을 조정하거나 인접 항목 사이를 보간한다.
4. 조회한 값을 나머지 계산에 사용한다.
5. 요구되는 정밀도와 표의 반올림 자릿수를 확인한다.

로그표는 이 원리를 일반적인 곱셈과 나눗셈에 널리 적용한 사례다. 두 수의 곱을 구할 때 각 수의 로그를 표에서 찾아 더한 다음, 결과에 대응하는 수를 다시 표에서 찾을 수 있다. 이 방식은 어려운 곱셈을 조회, 덧셈과 역조회로 바꾼다.

## 제작 과정과 오류

인쇄된 수학 표는 하나의 계산 결과물이면서 여러 단계가 연결된 제작 시스템이었다.

```text
문제와 수식 선정
→ 계산 작업 분해
→ 값 계산
→ [[검산]]과 비교
→ 전사와 편집
→ 활자 조판과 교정
→ 인쇄와 배포
→ 사용자의 조회와 보간
```

| 단계 | 발생할 수 있는 문제 |
|---|---|
| 계산 | 산술 실수, 잘못된 공식이나 초기값, 반올림 오류 |
| 검산·전사 | 숫자의 누락, 자리 바꿈, 부호와 소수점의 오기 |
| 조판·인쇄 | 활자 배열 오류, 교정되지 않은 오식의 복제 |
| 사용 | 잘못된 행·열 조회, 단위 혼동, 부적절한 보간 |

표는 계산을 반복할 필요를 줄이지만, 잘못된 값이 인쇄되면 같은 오류가 많은 사용자에게 반복 전달될 수 있다. 따라서 정확한 표의 제작은 개별 계산자의 숙련뿐 아니라 작업 분담, 독립적인 검산, 조판과 교정의 신뢰성에 달려 있었다.

## 배비지와 표 제작 자동화

배비지가 해결하려 한 문제는 단순히 계산을 빠르게 하는 것이 아니었다. 19세기의 항해, 천문학과 공학은 많은 수치가 정리된 표에 의존했고, 사람이 계산한 값을 옮겨 적고 인쇄하는 과정에는 여러 오류 지점이 있었다. 배비지는 계산과 인쇄 사이에서 사람이 숫자를 다시 옮기는 단계를 줄이려 했다.

[[차분 기관]]은 [[유한 차분법]]을 이용해 다항식으로 표현할 수 있는 값들을 반복적인 덧셈으로 계산하도록 설계되었다. 배비지는 1822년 발표에서 작은 기계로 제곱수·삼각수와 `x²+x+41`의 값 표를 만들었다고 보고하고, 계산 결과의 순서에 맞춰 활자를 배치하는 구상도 설명했다. 배비지가 직접 밝힌 효과는 계산된 수를 인쇄본으로 옮기는 과정의 오류를 막는 것이었다. 이를 계산 장치와 출력 장치를 연결해 제작 과정의 여러 오류 지점을 함께 줄이려 한 시도로 보는 것은 그 설계에 대한 해석이다.

이 관점에서 차분 기관은 특정 수식을 푸는 기계인 동시에 인쇄 가능한 수학 표를 생산하는 시스템이다. 이후의 [[해석 기관]]이 여러 절차를 실행하는 범용 프로그램 가능성으로 나아갔다면, 차분 기관의 출발점은 고정밀 표라는 사회적으로 사용 가능한 결과물을 안정적으로 만드는 일이었다.

## 컴퓨팅사적 의미

수학 표는 이미 수행한 계산을 저장해 이후의 계산 비용을 줄인다. 현대적인 표현을 빌리면 계산 시간을 저장 공간과 사전 준비로 바꾸는 방식이지만, 역사적 수학 표를 현대의 데이터베이스나 소프트웨어와 동일한 것으로 보아서는 안 된다.

컴퓨팅사에서 수학 표가 중요한 이유는 세 가지다.

- 계산 결과를 개인의 일회적 작업이 아니라 보존하고 배포할 수 있는 자원으로 만들었다.
- 계산, 검산, 기록, 인쇄와 사용을 하나의 정보 처리 과정으로 연결했다.
- 반복 계산과 오류 관리의 문제가 인간 계산 노동의 조직화와 기계적 자동화를 촉진했다.

배비지의 차분 기관은 이 세 요소 가운데 계산과 인쇄를 직접 연결하려 한 사례다. 따라서 수학 표는 기계식 계산 장치가 왜 필요했는지를 설명하는 배경일 뿐 아니라, 자동화가 생산해야 했던 구체적인 정보 산출물이었다.
